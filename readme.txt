A lightweight addon plugin for Elementor that provides a collection of custom, easy-to-use widgets.

== Description ==

Easy Elements is a powerful yet simple plugin that adds custom widgets to Elementor, giving you more control and flexibility in your page design. No need for bloat â€” just the essential building blocks you need to create beautiful layouts quickly.

**Features:**
* Lightweight and fast
* Clean, extendable code
* Includes a growing library of widgets
* Fully compatible with Elementor Free and Pro

Ideal for developers and designers who want to save time and stay in control.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Edit a page with Elementor and find the new Easy Elements widgets in the panel.

== Frequently Asked Questions ==

= Does this require Elementor Pro? =
No. It works with Elementor Free. Some features may enhance with Pro, but it's not required.

= Will this slow down my website? =
Absolutely not. Easy Elements is optimized for speed and only loads what's necessary.
